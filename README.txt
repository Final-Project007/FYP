# Student Retention Analysis

## Files
- Objective_Analysis.py --> models with GridSearchCV
- Subjective_Analysis.py --> smaller dataset analysis
- Dataset:
	games-features == Objective_Analysis.py
	questionnaire_responses == Subjective_Analysis.py


1. Install requirements:
   pip install -r requirements.txt

2. Run subjective model:
   python subjective_analysis.py

3. Run objective model:
   python objective_analysis.py
